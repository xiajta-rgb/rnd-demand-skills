# 研发需求智能治理技能 (RnD Demand Governance Skill)

一个专为研发部门打造的自动化需求评审与治理工具，基于公司《需求评审评分规则》，实现「需求输入→自动打分→定级分类→智能决策→材料生成」的全流程自动化。

## 功能特性

- **多维度自动评分** - 业务价值、战略匹配度、影响范围、投入产出比、紧急程度、实施复杂度
- **智能定级分类** - A级（战略型）/ B级（经营优化型）/ C级（日常改进型）/ D级（无效型）
- **自动生成材料** - 标准化提报材料 + 部门宣导文案
- **规则可配置** - 修改 YAML 文件即可调整评分逻辑
- **透明可解释** - 全程基于规则执行，无人工主观判断

## 快速开始

### 环境要求

- Python 3.7+
- PyYAML

### 安装

```bash
pip install pyyaml
```

### 使用方式

#### Python 代码调用

```python
from scripts.governance import execute

result = execute(
    demand_list=[
        {
            "id": "D001",
            "description": "跨部门订单数据同步系统开发",
            "submitter": "研发-张三",
            "usage_scenario": "供应链、销售、财务跨部门数据协同",
            "expected_value": "减少数据重复录入，提升30%对账效率",
            "cross_department_impact": "供应链、销售、财务3个部门",
            "resource_input": "2人月开发资源",
            "deadline_risk": "年度战略重点项目"
        }
    ],
    dept_submitter="研发部-小夏",
    is_trial_period=True,
    strategy_focus=["跨部门数据协同", "运营效率提升"]
)

print(result["预审总览"])
```

#### 命令行运行

```bash
cd rnd-demand-skills
python -m scripts.governance
```

## 评分机制

| 维度 | 权重 | 说明 |
|------|------|------|
| 业务价值 | 25分 | 收入提升/降本增效 |
| 战略匹配度 | 20分 | 与公司战略契合度 |
| 影响范围 | 15分 | 跨部门/多岗位受益 |
| 投入产出比 | 15分 | ROI评估 |
| 紧急程度 | 15分 | 合规/风险/客户影响 |
| 实施复杂度 | 10分 | 反向计分，越简单得分越高 |

| 总分 | 等级 | 类型 | 决策 |
|------|------|------|------|
| ≥80分 | A级 | 战略型需求 | 独立立项评审 |
| 60-79分 | B级 | 经营优化型需求 | 按价值排序评审 |
| 40-59分 | C级 | 日常改进型需求 | 轻量快速处理 |
| <40分 | D级 | 无效型需求 | 驳回并说明原因 |
